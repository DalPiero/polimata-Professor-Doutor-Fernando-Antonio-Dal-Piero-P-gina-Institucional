## Nome da página

**Professor Doutor Fernando Antonio Dal Piero — Página Institucional**

## Nome recomendado do repositório

Minha recomendação principal:

**`professor-dal-piero`**

## Estrutura dos arquivos

Coloque estes arquivos na mesma pasta do repositório:

- `index.html`
- `style.css`
- `script.js`
- `README.md`

## Pasta local sugerida no Windows

Use esta pasta:

`C:\Users\Windows\Downloads\GitHub Pages\professor-dal-piero\`

Dentro dela, salve:

- `index.html`
- `style.css`
- `script.js`
- `README.md`

## Como publicar no GitHub Pages

1. Entre em sua conta do GitHub.
2. Clique em **New repository**.
3. Nome do repositório: **`professor-dal-piero`**
4. Deixe como **Public**.
5. Clique em **Create repository**.
6. Envie os quatro arquivos para o repositório.
7. Vá em **Settings**.
8. Clique em **Pages**.
9. Em **Source**, selecione **Deploy from a branch**.
10. Em **Branch**, escolha **main** e **/root**.
11. Clique em **Save**.

## Endereço final da página

Se o seu usuário no GitHub for `DalPiero`, a página ficará assim:

`https://dalpiero.github.io/professor-dal-piero/`
